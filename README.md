# 个性化旅行规划与叙事生成助手

基于大语言模型的智能旅行规划系统，根据用户偏好、预算和天数生成个性化旅行计划。

## 功能特点
- 🎯 个性化旅行路线规划
- 💰 智能预算分配
- 🏨 酒店推荐（含模拟链接）
- 📖 生动的旅行叙事生成
- 🗺️ 每日详细行程安排

## 快速开始
1. 安装依赖：`pip install -r requirements.txt`
2. 配置API密钥：复制 `.env.example` 为 `.env` 并填写密钥
3. 运行应用：`streamlit run app.py`

## 技术栈
- 前端：Streamlit
- AI模型：智谱GLM-4
- 后端：Python
- 部署：Streamlit Cloud / 本地
